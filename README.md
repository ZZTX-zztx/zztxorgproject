# zztxorgproject

这是一个聚合型前端小工具项目，聚焦轻量
网页端工具的基础开发与搭建，目前已实现
FFmpeg网页版转换器、Hacker CMD命令行
仿真工具的初始前端结构，整体基于HTML
构建，模块独立、易拓展，适合后续迭代完
善交互与功能逻辑。

## 项目核心模块

### 1. FF 模块:

为FFmpeg网页版转换器专属目录，已完成index.html页面的基础搭建，是网页端实现音视频格式转换功能的核心载体，无需本地安装FFmpeg即可通过网页端操作。

### 2. hacker 模块:

针对Hacker CMD仿命令行交互工具开发，搭建了初始HTML基础结构，为后续实现命令解析、终端式交互反馈等功能提供了基础框架。

## 开发与拓展

项目各模块均以纯HTML为基础，可直接在现有结构上扩展CSS、JavaScript等技术，实现界面美化、交互逻辑编写与功能落地；后续可针对FF模块补充音视频选择、格式配置、转换执行等功能，为hacker模块增加命令解析、交互反馈、自定义命令等能力，也可基于此项目架构新增更多网页端实用小工具。
